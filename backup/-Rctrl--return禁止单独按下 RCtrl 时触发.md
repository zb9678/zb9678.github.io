## 如何使 Rctrl & a 快捷键组合按下时，不会激活 Send, #{Space}

- 给脚本添加   

` *Rctrl::return  ; 禁止单独按下 RCtrl 时触发`

```
Rctrl::  
Send, #{Space}
        SoundBeep, 300, 5
return
;ΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞ  切换中英输入法   Rctrl  ΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞΞ 4-75
```

以下为SnoMouse.ahk脚本中的一部分

```
*Rctrl::return  ; 禁止单独按下 RCtrl 时触发
; 鼠标运动处理
*>^a:: 鼠标模拟.左按("a")
*>^d:: 鼠标模拟.右按("d")
*>^w:: 鼠标模拟.上按("w")               ; * 号可以避免和系统自带的Alt+w冲突，造成上移时关闭文档。
*>^s:: 鼠标模拟.下按("s")
```
