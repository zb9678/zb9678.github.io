## 进入Program Files

```
C:\>cd \pro

按下Tab键

C:\>cd "\Program Files"

回车进入Program Files

C:\Program Files>

```

## 进入Program Files (x86)目录

```
C:\>cd \pr

按下Tab键

C:\>cd "\Program Files"

再次按下Tab键

C:\>cd "\Program Files (x86)"

回车进入Program Files (x86)目录

C:\Program Files (x86)>

```

## 转入其他目录　cd /d d:\3

注意：如果不加参数/d，是无法转入其他目录的。如
- d:\3>cd c:\
- d:\3>

```
C:\Program Files (x86)>cd /d d:\3

d:\3>

```

## 打开当前目录　start .

```
d:\3>

d:\3>start .

d:\3>

```
