> [!TIP]
>  ? 的作用

- 可以有前缀 y   如可以打开 yā   
     - - 没有 ？的话，就打不出yā，只能打出 ā

```
::a8::ā     a8        需空格ā       ya8   只能单独使用，前面有y就不能变为 ā
:?:a0::ā   a0        需空格ā        yā     前面有y没关系。
```
-  以上这2个实例中，要注意 ā 的后面都有一个空格

> [!TIP]
>   * 的作用

- 自动输出   （区别于 o）
            - - 去除 ā 后的空格

```
:*:a9::ā    无需空格 ā          ya9      只能单独使用，前面有y就不能变为 ā
:*?:a7::ā   无需空格 ā          yā       前面有y没关系。
```
-  以上这2个实例中，要注意 ā 的后面没有空格

> [!TIP]
>   o 的作用

- 去除 ā 后的空格
- - 无法自动输出（* 号则可以，注意区别）

```
::a7::ā      ā 
:o:a6::ā    ā
```
